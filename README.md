
# rx-base-react-native

React-Native base component

React-Native 常用的基本控件

## Getting started

`$ npm install rx-base-react-native --save`


## Usage
```sh

import {
  RXButton,

  RXDottedLine,

  RXScrollView,
  RXFlatList,

  RXText,
  RXTextInput,

  RXTouchable,

  RXTheme,

  RXPlatform,
  RXDate,
  RXNumber,
  RXString,
  RXColor,
  RANDOM_COLOR,
} from 'rx-base-react-native'

```
